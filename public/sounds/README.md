# Archivos de Sonido para BingUp

Coloca aquí los archivos de sonido en formato MP3.

## Archivos requeridos:

| Archivo | Descripción | Sugerencia |
|---------|-------------|------------|
| `ball-called.mp3` | Suena cuando se canta una balota (solo jugadores) | Sonido "ting" o campana ligera |
| `ball-draw.mp3` | Suena cuando el admin saca una balota | Efecto de máquina de bingo/ruleta |
| `mark.mp3` | Click al marcar un número en el cartón | Click satisfactorio |
| `bingo-called.mp3` | Cuando alguien canta bingo | Campana/alerta |
| `bingo-win.mp3` | ¡Ganaste! Fanfarria de celebración | Fanfarria corta |
| `bingo-false.mp3` | Bingo falso | Buzzer de error |
| `game-start.mp3` | Al iniciar la partida | Silbato o campana de inicio |
| `game-end.mp3` | Al finalizar la partida | Sonido de cierre |
| `card-shuffle.mp3` | Al cambiar el cartón | Barajar cartas |
| `player-join.mp3` | Jugador se une a la partida | Notificación sutil positiva |
| `player-leave.mp3` | Jugador se desconecta | Notificación sutil |
| `click.mp3` | Click genérico de UI | Click suave |
| `success.mp3` | Acción exitosa | Sonido de confirmación |
| `error.mp3` | Error genérico | Sonido de error corto |

## Fuentes gratuitas de sonidos:

- [Mixkit](https://mixkit.co/free-sound-effects/) - Sonidos gratuitos sin atribución
- [Freesound](https://freesound.org/) - Biblioteca colaborativa (verificar licencia)
- [Pixabay](https://pixabay.com/sound-effects/) - Libres de derechos
- [Zapsplat](https://www.zapsplat.com/) - Gratuitos con registro

## Notas:

- Formato recomendado: MP3 (mejor compatibilidad)
- Duración: Mantener los sonidos cortos (< 3 segundos)
- Si un archivo no existe, el juego funcionará sin sonido para ese evento
